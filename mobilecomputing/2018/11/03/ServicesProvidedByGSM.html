Services Provided By GSM

November-3-2018
aurimusblack

Mobile Services

GSM defined 3 diffrent categories of services:

  • Bearer
  • Tele
  • Supplementary

Reference Model:


  • The Mobile Station(MS) is connected to GSM Public Land Mobile Network(PLMN) via the Um interface.
  • This network is connected to the Trasnit Network ISDN / PSTN.
  • Which is then connected to the Mobile Terminal(TE).
  • There might be a source/destination in between.

Bearer Services

  • Comprises of all the services for transparent transmission of data between interfaces to network
    • Ex: S - MS , U,S,R - ISDN
  • In classic GSM transmission was connection oriented (packet / circuit switched).
  • They only need the lower three layes of OSI Model.
  • Permit the following types of data transfer:
    • Transparent
    • Non Transparent
    • Synchronous
    • Asynchronous
  • Transparent Data Transfer:
    • Only uses physical layer
    • Transmission has constant delay & throughtput
    • To increase transmission quality, It user Forward Error Correction(FEC) which encodes redundancy into data stream.
    • If error is encountered it simply reconstructs using redundant data.
    • Data rates of 2.4,4.8 and 9.6 kbits/s are possible
  • Non Transparent Data Transfer:
    • Use protocols of layer2 and layer 3 to implement error correction and flow control
    • Uses transparent data transfer + Radio Link Protocol(RLP)
    • RLP has mechanisms like high level datalink control HDLC, selective rejection to trigger retransmission of data
    • bit error rate is < 10-7

  • Synchronous and Asynchronous Data Transfer:

    • both provide full duplex communication.
    • Synchronous provides data rates of 1.2, 2.4, 4.8, 9.6 kbits/s
    • Asynchronous supports data rates from 3000 - 9600 bits/s

Tele Services

  • GSM mainly focuses on voice oriented services:
    • Encrypted voice transmission
    • message services
    • data communication to PSTN, ISDN
  • However the main service is telephony
  • GSM provides a feature called Emergency Number:
    • mandatory and free for all.
    • highest priority, when initiated every connection is premptied.
  • Another service provided by GSM is Short Message Service(SMS):
    • Support upto 160 characters
    • uses unused capacity in signalling channels for transmission
  • Extended Message Service is the succesor of SMS which supports larger transmission size and has support for formatted text, animated pics etc.
  • Multimedia Message Service offers transmission of larger pictures, short video clips etc.

Supplementary Services

  • Offer various enhancements for the standard telephony service:
    • user identification.
    • call redirection.
    • forwarding of ongoing calls.
    • support for closed groups.
